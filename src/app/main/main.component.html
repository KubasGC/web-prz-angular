Co to jest Web?

Web jest powszechnie używanym skrótem dla ogólnoświatowej sieci World Wide Web (WWW). Podstawą technologii Web jest protokół HTTP służący do przesyłania dokumentów hipertekstowych. Historia tworzenia stron internetowych wiąże się z upublicznieniem pierwszej specyfikacji języka HTML w roku 1991, której twórcą był Tim Berners-Lee. Ustanawianiem standardów technologii Web zajmuje się konsorcjum W3C.
